# Portfolio Animatsiyalar va Xususiyatlar

## Qo'shilgan Animatsiyalar

### 1. **Framer Motion Animatsiyalari**
- **Navbar**: Yuqoridan pastga slide animatsiya, hover effektlari
- **Hero Section**: Fade va slide animatsiyalar, floating user image
- **Buttons**: Scale va shadow hover effektlari
- **Cards**: Zoom, rotate va scale animatsiyalar
- **Skills & Projects**: Stagger animatsiyalar

### 2. **AOS (Animate On Scroll)**
- Scroll qilganda elementlar paydo bo'ladi
- Fade-up, zoom-in, flip-left animatsiyalar
- Har bir element uchun delay qo'shilgan

### 3. **React Scroll**
- Smooth scrolling navigatsiya
- Navbar linklar bo'limlarga olib boradi
- Contact button ham smooth scroll qiladi

### 4. **Qo'shimcha Komponentlar**

#### LoadingScreen
- Portfolio ochilganda loading ekrani
- Progress bar animatsiyasi
- 2.5 soniyadan keyin yo'qoladi

#### ScrollToTop Button
- 300px scroll qilgandan keyin paydo bo'ladi
- Yuqoriga smooth scroll qiladi
- Hover va tap animatsiyalari

#### ParticlesBackground
- Canvas-da animatsiyalangan zarralar
- Zarralar bir-biriga bog'lanadi
- Background uchun interaktiv effekt

### 5. **CSS Animatsiyalar**
- Gradient shimmer effektlari
- Pulse animatsiya (hero bg-color)
- Hover transformatsiyalar
- Custom scrollbar
- Smooth transitions

### 6. **Hover Effektlari**
- Cards: Scale, rotate, shadow
- Images: Scale va glow
- Buttons: Ripple effect
- Links: Color va gap o'zgarishi

## Ishlatish

```bash
npm run dev
```

Portfolio saytingiz endi juda interaktiv va professional ko'rinishga ega!
